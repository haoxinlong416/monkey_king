# 创建远程仓库教程：monkey.king

## 步骤 1：注册 GitHub 账号

## 步骤 2：创建新的远程仓库

登录GitHub 账号,点击右上角的“+”号，选择“New repository”。

![image-20240827165035996](image-20240827165035996.png)

## 步骤3：建本地的版本

进入，右键-Git Bash-输入“git init”初始化成一个Git可管理的仓库

命令：git status

绿字表示已add到Git仓库上的文件
